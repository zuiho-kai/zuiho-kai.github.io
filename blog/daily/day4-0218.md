# Day 4 — Agent 学会自己做事

> 2026-02-18 · M4 + M5

今天是质变的一天 — Agent 从"被动响应"变成了"主动行动"。

## M4：自主行为引擎

之前 Agent 只会在被 @ 或被唤醒时回复。现在它们有了自主行为引擎：

- 定时 tick 触发 → LLM 根据当前状态决策
- 该上班了自己去打卡
- 信用点不够自己去接悬赏
- 闲着没事找人聊天

关键修复：
- 模型切换逻辑优化
- Prompt 精简（减少 token 消耗）
- Reasoning fallback（推理失败时的降级策略）

验证：9 个 E2E + 10 个单元测试全绿。

## M5：城市经济全功能

在 M4 自主行为的基础上，把城市经济系统做完整：

- 记忆系统和城市经济全功能联调
- 真实 E2E 验证通过

## 小结

Day 4 之后，Agent 不再需要人类"推一下动一下"。它们会自己规划行动、自己赚钱花钱。这是从"聊天机器人"到"AI 居民"的关键一步。
